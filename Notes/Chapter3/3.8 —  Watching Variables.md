# 3.8 —  Watching Variables

Watching variables is the process of inspecting variable values while the program is executing. Since your `launch.json` includes `-enable-pretty-printing`, complex objects like standard library containers will be formatted neatly.

- **Hovering:** Hover your mouse cursor over a variable in the editor to quickly see its current value.
    
- **Locals (VARIABLES Panel):** VS Code automatically displays the values of local variables in the **VARIABLES** section docked on the left during a debug session.
    
- **Watch Window (WATCH Panel):** You can type the name of a variable or a simple expression into the **WATCH** section, and it will be evaluated each time a breakpoint is hit.
    
- **Debug Console:** You can type expressions into the Debug Console to have them evaluated once.

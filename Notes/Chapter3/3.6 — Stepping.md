# 3.6 — Stepping

Stepping commands allow you to execute your code statement by statement.

- **Starting the Debugger:** Press `F5` or navigate to Run > Start Debugging. Your current `launch.json` has `"stopAtEntry": false`, meaning the program will run completely unless it hits a breakpoint.
    
- **Step Into (`F11`):** Executes the next statement. If the statement is a function call, the debugger jumps into that function and pauses at the top.
    
- **Step Over (`F10`):** Executes the next statement, but if it is a function call, it executes the entire function without stopping and returns control to you on the next line.
    
- **Step Out (`Shift+F11`):** Executes all remaining code in the current function and returns control to you when the function finishes.
